Channel Delay
=============

Faust DSP for delay channels independently.

Build
=====

Standalone JACK application
---------------------------

```bash
faust2jack channelDelayStereo.dsp
```

or

```bash
faust2jaqt channelDelayStereo.dsp
```

LV2
---

```bash
make lv2
```

Etc
---

See Faust docs to more info.
